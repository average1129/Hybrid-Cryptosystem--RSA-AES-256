Hybrid Cryptosystem for FIle Transfer
This project implements a simple Hybrid Cryptosystem for file transfer using AES 128 and RSA 2048
Data Structures
Here are the data structures with brief descriptions:
 CAES_ctx
 CAES_parametersStructure holds 16 bit AES Key and 16 bit IV
 CCryptStructure holds padded buffer size, the buffer where actual encryption takes place , and RSA and AES parameters structures, and a message structure holding two mpz_ts
 CmessageStructure holds space to encrypt and decrypt data in the form of mpz_t
 Cprivate_key_RSAStructure holds 5 different mpz_t variables which form the private key for RSA encryption
 Cpublic_key_RSAStructure holds Modulus and globally fixed Public Exponent both are mpz_t
 CRSA_parametersStructure holds both the private and public key of an encrypted message
